== Description ==

The plugin provides options to hide featured images from all posts and pages. 

= Features =

Hide featured images from all posts and pages.
Hide Admin Toolbar on user's end.
Remove the “Howdy” text in the upper right corner of your admin dashboard.
Remove the WordPress logo from the upper left corner of the admin bar.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Search` and search for WP Hide Show Featured Image).
* OR, You can also [download](https://downloads.wordpress.org/plugin/wp-hide-show-featured-image.zip "Download WP Hide Show Featured Image") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.

== Frequently Asked Questions ==

= How we can hide featured images from all posts or pages? =

Go to 'Wp Hide Show featured image' setting page ( `Plugins > Settings > WP Hide Show Featured Image`) and select the option(`YES/NO`) to wp hide show featured image for all posts or pages.

= Does this plugin hide featured images from custom post types? =
No, this feature is not available in this version.

= How to hide / show admin toolbar on the user end? =

Select the option (YES / NO) to hide or show the admin toolbar on the user end and press submit.

= How to remove the “Howdy” text in the upper right corner of your admin dashboard? =

Select the option (YES / NO) to remove the “Howdy” text in the upper right corner of your admin dashboard and press submit.

= How to remove the WordPress logo from the upper left corner of the admin bar? =

Select the option (YES/NO) to remove the WordPress logo from the upper left corner of the admin bar.
